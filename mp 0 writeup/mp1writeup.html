MP-1 Writeup

Overview and Usage:

Pomogranate is a Pomodoro timer that keeps you focused on your work by showing you pomegranates every time you open a social media site.

How to use this extension: Open link below > Code button > Download ZIP > Open .zip file in files/finder > Go to Google Chrome and type chrome://extensions > Toggle Developer Mode to be ON > Load unpacked > Select "pomogranate-extension-main" > Click puzzle icon in top right toolbar > You're ready to use Pomogranate!

Click here to access the GitHub link


Development Process:

1. I created a GitHub folder for my extension with a README file

2. I cloned the GitHub folder and opened it in VS Code

3. Created a manifest.json file for the Chrome extension

4. Created Content.js to block specific websites when accessed

5. Created popup.js and popup.html to create the Pomogranate extension popup with the timers

6. Added favicon and icons of different sizes

7. Commited changes to GitHub

Issue Deep-Dive:
One issue I encountered at the beginning was a logic error in which I was not able to start the timer when I first loaded the extension. Just the HTML showed up, but none of my CSS or JavaScript worked.

I solved it by looking over other examples of extensions to see how all the JavaScript and CSS files interacted. I realized the HTML file had a line of code at the top: "link rel="stylesheet" href="./.....css" which linked the HTML to the CSS file. Furthermore, I added a line of code that had "script" on both sides, which would essentially tell the HTML file to execute the specified JavaScript file. This made me understand that the HTML, CSS, and JavaScript files all need to communicate to each other, so I will remember to put those lines from now on into my code.
