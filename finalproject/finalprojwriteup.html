Final Project Writeup

Overview and Usage:

ArtPad2.0 is an online drawing app where a user can draw using their mouse. Users can select a pen color by clicking the color setting

How to use:

Click on the color boxes under "Pen Color" and "BG Color" to select colors for the respective choices.

Adjust brush size using the slider.

Draw anything by clicking and dragging across the screen. Undo or clear as needed.

Download your work and give it a name if you'd like.

Issue Deep-Dive:
One issue I encountered was that after deploying my website, the CSS would not show up despite it showing up on npm run start.

In order to fix this, I had to add "style.css" to my rollup.config.js under the src section, next to "index.html," as I think the web could not find my CSS file if I didn't write it into the rollup file.
