Cheatsheet

CSS

  • Change Font Color on Hover: header:hover{color: #hex code;}
  • Creating Class: class name {insert stuff here (i.e. colour, margin, etc.)}
  • Linking CSS to HTML (remove brackets): [<]link href="styles/style.css" rel="stylesheet"[/>]

HTML

  • Change Font Color on Hover: header:hover{color: #hex code;}
  • Basic format:
    [<]!DOCTYPE html[>]

    [<]html[>]

    [<]body[>]

    {insert words here}

    [<]/body[>]

    [<]/html[>]

  • Images: [<]img[>] -> src, alt
  • Links: [<]a href="url">link text[<]/a[>]
  • Lists: [<]ul class="list class name"[>] [<]li[>] insert list item [ [<]/ul>

JavaScript

  • console.log()
  • prompt("Message") will prompt for user input -> null if cancelled
  • Inputs: [<]input type="input type" [<]input/>
  • document.getElementById('id')

Deploying a Site

  • Create and clone Git repository
  • Run npm init
  • Install modules
  • Create index.html, index.js, and style.css (optional)
  • npm install --save-dev @web/dev-server
  • Under scripts in package.json: "start": "web-dev-server --node-resolve --open --watch"
  • npm run start
  • npm install rollup --save-dev
  • npm install @rollup/plugin-node-resolve --save-dev
  • npm install rollup-plugin-copy --save-dev
  • Create rollup.config.js: import { nodeResolve } from "@rollup/plugin-node-resolve"; import copy from "rollup-plugin-copy"; module.exports = { input: "index.js", output: { dir: "dist", }, plugins: [ copy({ targets: [{ src: "index.html", dest: "dist" }], }), nodeResolve(), ], };
  • Package.json under scripts: "build": "rollup --config"
  • npm run build
  • Create .gitignore and add node_modules/ and dist/
  • Add, commit, and push changes
  • npm install gh-pages --save-dev
  • Package.json under scripts: "deploy": "npm run build && gh-pages -d dist"
  • npm run deploy
